# Fun with fractals
Java swing desktop application for generating different types of fractal images.

Types of fractals it can generate:

#####Mandelbrot set
#####Julia set
#####Newton fractals
#####Barnsley's fern
#####Dragon curve
#####Pythagora's tree
#####Brownian tree
#####Fractal terrain
#####Sierpinski's gasket
#####Fractal flames
